[백준링크](https://www.acmicpc.net/problem/11726)


다이나믹 프로그래밍 

# 파이썬
```python
n = int(input())
dp = [0 for i in range(n+1)]
if n<=3 :
    print(n)
else:
    dp[1] = 1
    dp[2] = 2
    for i in range(3,n+1):
        dp[i] = dp[i-1]+dp[i-2]
    print(dp[i]%10007)
```
