[문제링크](https://www.acmicpc.net/problem/11055)

DP 활용하기

data 값 입력 받고 비교를 통해 DP에 값 저장

# 파이썬
```python

import sys
input = sys.stdin.readline

n = int(input())
data = list(map(int,input().split()))
dp = [i for i in data ]

for i in range(n):
    for j in range(i):
        if data[j] < data[i]:
            dp[i] = max(dp[i], dp[j]+data[i])
print(max(dp))
        

```
