[문제링크](https://www.acmicpc.net/problem/2108)

사전형 함수 사용

# 파이썬
```python
import sys
input = sys.stdin.readline
n = int(input())
s=[]
for _ in range(n):
    s.append(int(input()))

s.sort()
print(round((sum(s)/len(s))))
print(s[(len(s)//2)])
dic = dict()
for i in s:
    if i in dic:
        dic[i] +=1
    else:
        dic[i] = 1
a = max(dic.values())
a_dic = []
for i in dic:
    if a == dic[i]:
        a_dic.append(i)

if len(a_dic) >1:
    print(a_dic[1])
else:
    print(a_dic[0])
print(max(s) - min(s))
```
